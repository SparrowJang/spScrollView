sp-scroll-view
===================

Swtich views by drag and drop.

## Install dependencies

```bash
bower install
npm install
```

## Usage

```html
<sp-scroll-view>
  <div>Hello world</div>
  <div>Hello all</div>
</sp-scroll-view>
```

## method

**moveByIndex : Number**
> move scroll

```js
spScrollView.moveByIndex( 1 )
```

**getViews**
> get all views of child

```js
spScrollView.getViews()
```

## DEMO

```bash
gulp server
```
